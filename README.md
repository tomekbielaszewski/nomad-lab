# Nomad Lab

I will learn Nomad in this repo. I'm planning to use Vagrant to set up a bunch of VMs, install Nomad there and run some 
workflows.

## Useful commands

`vagrant up`  
`vagrant list-commands`  
`vagrant status`  
`vagrant halt`  
`vagrant ssh nomad-node-1`  

`nomad node status`
